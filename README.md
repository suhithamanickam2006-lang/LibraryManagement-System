*Library Management System*
Overview:
The Library Management System is a Java-based application developed using IntelliJ IDEA and MySQL.
It helps manage library operations such as adding books, viewing available books, issuing books, and returning books. The project uses JDBC for database connectivity and follows Object-Oriented Programming (OOP) principles.

Features:
->Add new books to the library
->View all books stored in the database
->Issue books to users
->Return issued books
->Store book records permanently using MySQL
->Database connectivity using JDBC
->Technologies Used Java,IntelliJ IDEA,MySQL,JDBC (Java Database Connectivity) and Object-Oriented Programming (OOP).
